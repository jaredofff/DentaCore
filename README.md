# 🦷 DentaCore

Plataforma integral de gestión clínica odontológica desarrollada con **Next.js 16**, **Supabase** y **Tailwind CSS**.

## 🚀 Funcionalidades Actuales
- **📁 Gestión de Pacientes**: Registro completo con generación de folios automáticos.
- **🦷 Odontograma Interactivo**: Esquema dental FDI funcional con seguimiento por pieza.
- **📝 Historial Clínico (SOAP)**: Notas de evolución estructuradas y detalladas.
- **💊 Recetario Digital**: Gestión de prescripciones y farmacoterapia.
- **📸 Galería Radiográfica**: Carga y visualización de estudios diagnósticos.
- **📅 Agenda Clínica**: Gestión de citas y estados (Pendiente, Completada, Cancelada).

## 🛠️ Stack Tecnológico
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS.
- **Backend/DB**: Supabase (PostgreSQL, Auth, Storage, RLS).
- **UI/UX**: Lucide React, Diseños con jerarquía visual clínica.

## 📦 Despliegue en Vercel
1. Conecta este repositorio a Vercel.
2. Configura las variables de entorno `NEXT_PUBLIC_SUPABASE_URL` y `NEXT_PUBLIC_SUPABASE_ANON_KEY`.
3. ¡Listo! La aplicación se desplegará automáticamente.
